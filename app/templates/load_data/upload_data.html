Information

Dataset is mandatory, it must be a zip file containing indivdual recordings. Label set is optional. If label is uploaded the number of row must match the number of recordings in the dataset. Attention, files in the dataset are sorted by their name in the alphabetic order, the rows of the label set must be ordered accordingly. Here are the files specification:

  • Dataset:
    • Must be a zip file
    • Each file must contain a one dimensional signal
    • Positive value correspond to inhalation phase and negative to exhalation
    • Each signal can be of the following format:
      • .text, .csv: the delimiter is the line break (\n)
      • .EDF: only the first canal will be take in consideration
  • Label set:
    • Must be a .txt or .csv: the row separator is the break line (\n) and the column separator is the coma (,).
    • The first row corresponds to the labels
    • Do not include index column it wil be generated
